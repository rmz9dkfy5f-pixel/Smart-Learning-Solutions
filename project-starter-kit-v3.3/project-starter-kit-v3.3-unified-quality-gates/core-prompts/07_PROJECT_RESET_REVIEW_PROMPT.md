# 07 — Project Reset Review Prompt

Use this when the project gets messy.

```md
Perform a project reset review.

Do not write code.

Analyze:

1. current project goal
2. current working state
3. what is broken
4. what is unfinished
5. what is overbuilt
6. what can be deleted
7. what can be deferred
8. current source-of-truth conflicts
9. next smallest vertical slice
10. checks that must pass before continuing

Recommend one:

- continue
- repair
- simplify
- restart
- abandon

Do not make changes until approved.
```
