# 03 — First Task Plan Prompt

Use this to plan the first vertical slice.

```md
Plan only the first vertical slice.

Do not implement.

The slice must be small enough to verify quickly.

Include:

1. slice name
2. purpose
3. validation goal
4. user/operator-facing outcome
5. systems touched
6. temporary vs final work
7. success criteria
8. out-of-scope items
9. risks and assumptions
10. expected files to change
11. initial check command
12. verification command or manual test
13. rollback plan
14. approval request before implementation

Also update the proposed entries for:

- PLAN.md
- PHASE_GATES.md
- BACKLOG.md
- PROGRESS_NOTE.md
- COMMIT_NOTES.md

Do not edit files until approved.
```
