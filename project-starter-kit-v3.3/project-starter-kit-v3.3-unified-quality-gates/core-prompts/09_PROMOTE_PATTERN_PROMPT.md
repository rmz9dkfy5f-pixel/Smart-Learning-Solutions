# 09 — Promote Pattern Prompt

Use this after solving a repeated problem.

```md
Review the completed session and decide whether any reusable knowledge should be promoted.

Do not write code.

Identify:

1. repeated problem pattern
2. reusable solution
3. when to use it
4. when not to use it
5. secure defaults
6. tradeoffs
7. related files
8. whether this belongs in ai/patterns or ai/prompts

If useful, produce a clean Markdown file suitable for:

ai/patterns/pattern_name.md

or:

ai/prompts/prompt_name.md
```
