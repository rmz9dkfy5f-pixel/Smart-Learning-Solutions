# 05 — Phase Gate Review Prompt

Use this before moving from one phase to the next.

```md
Perform a phase gate review.

Do not implement anything.

Compare current state against the phase gate criteria.

Report:

1. what passed
2. what failed
3. what is incomplete
4. what changed from the plan
5. what risks remain
6. what should be cut from scope
7. whether we should continue, fix, simplify, or revise
8. recommended next vertical slice

Update recommendations for:

- STATUS.md
- PHASE_GATES.md
- SLICE_REVIEWS.md
- PROGRESS_NOTE.md
- PROGRESS_NOTES.md
- LESSONS_LEARNED.md

Do not make changes until approved.
```
