# PRE-BUILD READINESS

Answer these before serious implementation:

- [ ] Do we know the exact problem being solved?
- [ ] Do we know who the primary user is?
- [ ] Do we know what is out of scope?
- [ ] Do we have primary user flows?
- [ ] Do we know the core entities and relationships?
- [ ] Do we know the major states and transitions?
- [ ] Do we know validation and permission rules?
- [ ] Do we know the technical boundaries?
- [ ] Do we have acceptance criteria?
- [ ] Do we know what counts as version bump vs schema bump?
- [ ] Do we know the first vertical slice to build?

If any answer is "no", planning is incomplete.
