Acceptance Criteria
This homepage is successful if:
* a new visitor understands in under 5 seconds that this is a STEM workshop provider
* the primary CTA is unmistakably Book a Workshop
* the two programs are clear and visually balanced
* the page feels premium/professional, not template-like
* the Resources area supports trust but does not dominate the site
* dead pages from the current site are eliminated from the experience
* the homepage can be built cleanly as raw source code without requiring ongoing CMS operations
