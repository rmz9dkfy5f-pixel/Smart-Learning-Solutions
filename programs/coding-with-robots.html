Coding with Robots

Students build, program, and test real robots — learning logic, creativity, and engineering thinking through hands-on experience with the Edison robotics platform.

Four Coding Levels

From First Commands to Python

Our program grows with your students — from simple visual commands to real Python programming, each level builds naturally on the last.

Level 1

Barcode Lessons

The entry point — students use pre-printed barcodes to issue commands to their Edison robot. No prior experience needed.

Level 2

EdBlocks

Visual block-based programming. Students drag and connect blocks to build sequences, loops, and logic — intuitive for any age.

Level 3

EdScratch

A bridge between visual and text-based coding. Students write real code syntax while still working in a supported environment.

Level 4

EdPy (Python)

Full Python programming for Edison. Students write, test, and debug real programs — genuine computational thinking at every step.
